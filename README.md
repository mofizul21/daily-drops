## Features
* Dynamic content feed with trending posts  
* User profile creation and customization  
* Like, dislike, and interactive engagement tools  
* Email authentication (Register & Login)  
* Real-time data sync with Cloud Firestore  
* Secure push notifications  
* Smooth and responsive UI experience  

## Tech Stack
*   **Frontend:** Flutter
*   **Backend:** Firebase
    *   Firebase Authentication
    *   Cloud Firestore
